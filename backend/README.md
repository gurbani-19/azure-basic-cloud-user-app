# Cloud User App 🚀

Full-stack cloud application deployed on Azure VM.

## Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js, Express
- Database: MongoDB (Docker)
- Server: Nginx
- Process Manager: PM2

## Features
- Add users
- Fetch users
- Persistent database (MongoDB)
- REST API

## Architecture
Frontend → Nginx → Node API → MongoDB

## Deployment
Hosted on Azure Virtual Machine
